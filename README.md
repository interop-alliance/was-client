## Install

- Node.js 24+ is recommended.

### PNPM

To install via PNPM:

```
pnpm install @interop/isomorphic-lib-template
```

### Development

To install locally (for development):

```
git clone https://github.com/interop/isomorphic-lib-template.git
cd isomorphic-lib-template
pnpm install
```
